## Instalacion

### 1. Clonar el repositorio

``` bash
git clone https://github.com/urielgudinouri-source/pfd.git
```

### 2. Instalar dependencias

``` bash
npm install
```

### 3. Configurar variables de entorno

Crea un archivo `.env` a partir del archivo de ejemplo:

``` bash
cp .env.example .env
```

### 4. Ejecutar proyecto
``` bash
npm run start
```
